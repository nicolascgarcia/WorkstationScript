# WorkstationScript
Script de automatização de instalações de softwares que utilizo para o OS Debian/Ubuntu Linux.

Softwares a serem instalados pelo script:

    - Google Chrome;
    - FileZilla;
    - Postman;
    - Skype;
    - VS Code;
    - Git;
    - Node.js;
    - Apache;
    - MySQL;
    - PHP 7.4;

Para utilizar o Script, basta rodar o comando:

sudo ./my-workstation.sh
